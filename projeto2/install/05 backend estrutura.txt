INICIAR O SERVIÇO NA PASTA BACKEND
npm init -y

INSTALAR O EXPRESS
npm install express

CRIE O ARQUIVO SERVER.JS NA PASTABACKEND COM O CONTEUDO
const express = require('express');
const app = express();
const PORT = 3000;

app.get('/', (req, res) => {
  res.send('Servidor rodando com sucesso!');
});

app.listen(PORT, () => {
  console.log(`Servidor iniciado em http://localhost:${PORT}`);
});

TESTE DO SERVIDOR
node server.js

TESTE DO SERVIDOR NO NAVEGADOR
http://localhost:3000/
