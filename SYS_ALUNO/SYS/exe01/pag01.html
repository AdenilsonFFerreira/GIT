Exercicio 01 de materia Desenvolvimento WEB 01



O desenvolvimento Web é o ramo da informática que estuda o desenvolvimento de soluções na web.
Incluindo o design, a usabilidade, funcionalidade, segurança e integração de back-end.

Sendo assim, pode-se dizer que envolve tudo o que é exibido na parte externa de um site.
Como a Exibição comunica com os servidores e software da aplicação e bancos de dados.

Desse modo, o desenvolvimento Web pode ser utilizado para desenvolver.
Sites, web applications, aplicativos móveis ou qualquer outra coisa que necessite estar presente na web.

Sendo assim, normalmente, o desenvolvimento Web começa com o desenho de interfaces e experiências do usuário.
A construção de telas usando HTML, CSS e outras tecnologias de programação da web e o envolvimento de outras funcionalidades.
Como frameworks ou bibliotecas de terceiros.
